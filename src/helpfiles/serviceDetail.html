Service Detail:

The service detail view is an additional view


Service Detail View shows some common service information.

It also shows you the last incidents from the log file.
Change Log line limit to view longer time range.

Double Click on the incident view selects the nearest log entry for you.
Log entries are collapsed by state change.

- Open in Browser
Opens this service in the browser, the url will be build from the
CGI path you configured in "Server Config". Whassup will append to
this cgi path an additional
"extinfo.cgi?type=2&host=Hostname&service=Description"

This should be fine for many monitoring system, if this does not
suite your requirements, please do not hesitate to contact me.
Alternativ you could take a look into mod_rewrite or
html meta refresh/redirection, if you like to use this information
in other parts.

Open Action URL and Notes Url will open your default browser and load
the full action/notes URl configured in your monitoring server.
