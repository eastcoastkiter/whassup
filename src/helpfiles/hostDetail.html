Host Detail View:

The host detail view is an additional view.

Host Detail View shows some common host information.

It also displays your the parent/child relations.
You can move any object in the parent/child view.

- Generation
Use Generation to add child/parent generation.

- Force
Layout your generations automatically

- Scale
Scale to all items in the scene

Double Click on a host will open Host Detail for that host.

Services:
Table of all services on this host. Double click to open
service detail for this host.

- Open in Browser
Opens this host in the browser, the url will be build from the
CGI path you configured in "Server Config". Whassup will append to
this cgi path an additional "status.cgi?host=Hostname".

This should be fine for many monitoring system, if this does not
suite your requirements, please do not hesitate to contact me.
Alternativ you could take a look into mod_rewrite or
html meta refresh/redirection, if you like to use this information
in other parts.

Open Action URL and Notes Url will open your default browser and load
the full action/notes URl configured in your monitoring server.
