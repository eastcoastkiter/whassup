ConfigWidget:
Config:
- Server Config:
Configure your connection to the monitoring server

- Refresh
How many seconds before ServiceWidget will refresh

- Timezone
Add or subtract hours to your monitoring server
if your local time differs

- Log Lines:
How many lines to search in Log Window

- Filter:
current Filters applied for Service Window
Please take a look into Filters documentation

- Font:
Selected font for all tree views

- additional Columns:
additinal Columns are shown in service view

- Hide on Close
hides Whassup in Systemtray instead of exiting

- Blinking Systemtray
let the systemtray blink, if new incidents are found

- Sort Service by State
Bydefault service will be sorted by name in service tab
Check this to sort by state

- Browser Url
add Server and Host URL for "Open Browser Button" %HOST% and %SERVICE%
will be replace by the actual host/service

Configuration will be written to local %APPDATA%
for example: C:\Users\User\AppData\Local\Monitoring\whassup\whassup.ini
